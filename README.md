# 📚 Smart Study Planner Generator

A futuristic and responsive web application that helps students create personalized study schedules based on subjects, exam dates, study hours, and difficulty levels.

## 🚀 Features

* Personalized Study Plan Generation
* Daily & Weekly Study Goals
* Smart Subject Prioritization
* Exam Countdown
* Progress Tracker
* Pomodoro Timer (25/5 Focus Mode)
* Motivational Quotes
* PDF Export & Print Support
* Background Music Integration
* Custom Project Poster Support
* Local Storage Data Persistence
* Fully Responsive Design
* Neon Digital Library Theme

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Tailwind CSS
* Local Storage

## 📸 Project Preview

Add your project poster as:

```text
poster.png
```

and place it in the root project folder.

## 🎵 Background Music

Add your music file as:

```text
music.mp3
```

and place it in the root project folder.

## 📂 Project Structure

```text
Smart-Study-Planner/
│
├── index.html
├── poster.png
├── music.mp3
└── README.md
```

## ▶️ How to Run

1. Download or clone the repository.
2. Place `poster.png` and `music.mp3` in the project folder.
3. Open `index.html` in your browser or use VS Code Live Server.
4. Generate your personalized study plan.
